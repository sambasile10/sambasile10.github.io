Projects

Twitch Tracker

Twitch Tracker was a project I made that recorded which streams are being watched by which users for a month straight. My data scraper was made in Typescript and used PostgreSQL for storage. After a month of data collection, community detection and clustering algorithms were used to create a ‘map’ of the website, where streams that had high overlaps in viewership are clustered together.
Repository | Paper | Full Size Map

Twitch Logger

Twitch Logger was a project that records and stores messages sent on predefined Twitch chat channels and allows for them to be searched later on. It consists of a Typescript backend for recording and storage using MongoDB as well as a web-based frontend written in React for searching through messages.
Repository

PlaylistPal

PlaylistPal is a work-in-progress project that allows for the transferring of music playlists between different streaming platforms (YouTube, Apple Music, and Spotify). Users can easily select a source platform and a playlist on it alongside a destination platform and PlaylistPal will do its best to transfer the playlist. It gives users the option to do manual corrections of songs it may have missed. It has a Java Spring backend and a web-based frontend written with Svelte.
